# Vaccine Marketing Strategy: N1H1

## Overview
This project focuses on optimizing vaccine outreach and resource allocation during the N1H1 rollout. The analysis identifies which populations should be prioritized and how limited resources can be deployed most effectively.

## Business Question
Which population segments should be targeted to maximize vaccine uptake, and how should outreach resources be allocated?

## Approach
- Analyzed demographic and behavioral data related to vaccine uptake
- Built classification models to identify high-probability adoption groups
- Evaluated targeting strategies using model outputs
- Translated findings into actionable marketing recommendations

## Tools & Techniques
- Python
- Pandas, NumPy
- Scikit-learn
- Classification models with class imbalance (SMOTE)
- Target marketing analysis

## Key Takeaways
- Targeted outreach significantly improves program efficiency
- Data-driven segmentation supports equitable and effective public health interventions
- Highlights the role of analytics in public health decision-making

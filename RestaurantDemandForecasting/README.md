# Restaurant Demand Forecasting

## Overview
This project analyzes restaurant-level business data to forecast customer demand and identify trends that impact operational performance. The analysis focuses on helping restaurants better allocate resources and optimize staffing and inventory decisions.

## Business Question
How can historical sales and customer behavior data be used to predict demand and improve restaurant operations?

## Approach
- Queried and cleaned transactional data using SQL
- Conducted time-based and customer-level analysis in Python
- Identified demand patterns and peak periods
- Translated findings into operational recommendations

## Tools & Techniques
- Python
- SQL
- Pandas
- Time series and trend analysis

## Key Takeaways
- Clear demand patterns emerged across time and customer segments.
- Forecasting insights can reduce waste and improve staffing efficiency.
- Demonstrates how analytics can directly inform day-to-day business decisions.

# Pneumonia X-ray Detection

## Overview
This project applies deep learning techniques to classify chest X-ray images for pneumonia detection. Medical imaging presents unique challenges, including class imbalance, image variability, and high stakes for decision accuracy.

The goal of this project was to explore how convolutional neural networks (CNNs) can support clinical decision-making while acknowledging the limitations of automated diagnosis.

## Clinical Question
How effectively can deep learning models assist in identifying pneumonia from chest X-ray images?

## Analytical Approach
- Preprocessed and normalized medical imaging data
- Built and trained a convolutional neural network (CNN)
- Evaluated model performance using classification metrics
- Analyzed strengths and limitations of the model in a clinical context

## Tools & Techniques
- Python
- TensorFlow / Keras
- Convolutional Neural Networks (CNNs)
- Medical image preprocessing

## Key Insights
- CNNs can effectively learn visual patterns associated with pneumonia
- Model performance highlights both potential and risk in clinical AI applications
- Demonstrates applied deep learning in a healthcare context

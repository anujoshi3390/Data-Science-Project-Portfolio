# Anu Joshi's Data Science Portfolio

Welcome! This repository contains a collection of my data science projects demonstrating applied analytics, machine learning, and data-driven decision-making. Each project has its own folder with code, data (if permitted), and documentation.

## Portfolio Projects

### 1. Boston House Price Prediction
Leveraged machine learning and regression modeling to forecast housing prices, uncover key price drivers, and enhance investment insights.  Techniques include VIF analysis, log transformations, Ridge/Lasso/Random Forest/CatBoost models, and GridSearchCV optimization.
**Tools:** Python, Scikit-learn, Pandas, NumPy, Matplotlib  
**Folder:** `BostonHousePricePrediction/`

### 2. Automotive Pricing Analysis

**Folder:** `AutomotivePricingAnalysis/`

### 3. Business Data Analysis: Restaurant Demand Forecasting

### 4. Vaccine Marketing Strategy: N1H1
